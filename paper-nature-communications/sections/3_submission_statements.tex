The authors jointly conceived the study, designed the analyses, performed the

\section*{Data availability}

All datasets are public datasets cited in the manuscript or generated analysis artifacts from controlled experiments. The Supplementary Information lists the run directories, aggregate tables and figure inputs. A public archival release will be provided on publication.

\section*{Code availability}

Analysis code, configurations, figure scripts and data-preparation instructions are available at \href{https://github.com/ToughClimb/false-science-induction}{github.com/ToughClimb/false-science-induction}. Public datasets are not redistributed in the code repository; the repository provides scripts and instructions for downloading or placing them at the paths used by the reproducibility configs.
